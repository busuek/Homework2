Заголовок	Расшифровка	Значение
Host	Hostname (or IP)	Имя узла (маршрутизатора) или его IP-адрес, через который проходит трафик.
Loss%	Packet Loss	Процент потерянных пакетов на пути до этого узла. Важный показатель проблем.
Snt	Sent	Общее количество отправленных пакетов.
Last	Last RTT	Время отклика (RTT) для самого последнего отправленного пакета (в миллисекундах).
Avg	Average RTT	Среднее арифметическое время отклика для всех пакетов.
Best	Best RTT	Минимальное зафиксированное время отклика для этого узла.
Wrst	Worst RTT	Максимальное зафиксированное время отклика для этого узла.
StDev	Standard Deviation	Стандартное отклонение времени отклика. Показывает, насколько сильно "скачет" задержка (джиттер). Чем меньше значение, тем стабильнее канал.
