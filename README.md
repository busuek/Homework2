3. Назначение ключа -f (Flood ping)
Ключ -f (от англ. flood — наводнение) переводит ping в так называемый "flood-режим". В этом режиме:

Утилита отправляет следующий пакет либо немедленно после получения ответа, либо 100 раз в секунду, в зависимости от того, что происходит чаще.

Для каждого отправленного запроса в терминале печатается точка (.), а для каждого полученного ответа точка стирается (возврат на шаг), что создаёт визуальную картину потери пакетов.

Важно: Это агрессивный режим, создающий большую нагрузку на сеть. Используйте его только для тестирования соединения между вашими виртуальными машинами c7-1 и c7-2.

Пример команды (выполняется на одной из ВМ, в качестве цели — адрес другой):

bash
sudo ping -f <IP-адрес-другой-ВМ>
