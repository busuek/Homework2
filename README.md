Основные ключи утилиты ping, которые вам понадобятся для работы, собраны в таблице ниже:

Ключ	Описание	Пример использования
-c	Отправить строго определённое количество пакетов и завершиться.	ping -c 5 192.168.100.13
-i	Задать интервал между отправкой пакетов в секундах.	ping -i 10 192.168.100.13
-s	Задать размер отправляемого пакета в байтах (без учёта заголовков).	ping -s 1500 192.168.100.13
-f	Режим "flood ping" (пинг-флуд) — отправка пакетов с максимальной скоростью.	ping -f 192.168.100.13
